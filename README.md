# 🤖 RAG PDF Chatbot

An AI-powered PDF Question Answering chatbot built using Retrieval-Augmented Generation (RAG).  
Upload a PDF and ask questions based on its contents.

## 🚀 Features

- 📄 PDF Upload Support
- ✂️ Text Chunking using RecursiveCharacterTextSplitter
- 🧠 Hugging Face Embeddings
- 🔎 Semantic Search with FAISS
- 🤖 Mistral AI Integration
- 💬 Conversational Question Answering
- 🎨 Streamlit User Interface
- 📚 Context-Grounded Responses using RAG

---

## 🛠️ Tech Stack

- Python
- LangChain
- Streamlit
- FAISS
- Hugging Face Embeddings
- Mistral AI
- RAG (Retrieval-Augmented Generation)
